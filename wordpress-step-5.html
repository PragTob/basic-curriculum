Step 5: Finetune the routes

If you try to open http://localhost:3000 it still shows the default page. Let’s make it redirect to the attendees page. On OS X and Linux, in the Terminal run: 
rm public/index.html
In Windows, in the Command Prompt run: 
del public\index.html
Then open config/routes.rb and add the following after the first line: 
root :to => redirect("/attendees")
